Endgame mítico

El panteón no aparece por accidente.

Cada jefe exige un invocador específico, una preparación distinta y devuelve reliquias que cambian cómo se juega.

El códice se alimenta desde la API de Odysseia: invocación, dificultad, combate y drops se mantienen separados de la decoración.

Cómo funciona

Una invocación, una pelea, una recompensa.

Antes de entrar, revisa la ruta exacta de cada encuentro. El sistema no usa un huevo genérico ni spawns aleatorios.
