# DrakesCraft Web Portal

The official web portal and store integration platform for **DrakesCraft Network**.

## Features

- 🌐 Modern web interface for server status, online players, and community statistics.
- 🛒 Seamless Tebex store & LuckPerms rank integration.
- 🎨 Custom branding matching DrakesCraft's visual identity.

## Tech Stack

- HTML5 / CSS3 / JavaScript (Vanilla)
- Node.js Backend API
- Tebex & Discord Webhooks
